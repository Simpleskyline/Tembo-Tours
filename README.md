**🐘 Tembo Tours & Travel**

---

A full-stack tour booking and travel management platform built with React, Django, and MySQL. This application allows users to explore destinations, view tour packages, and book accommodations through a responsive, modern interface.

---

**🚀 Features**
--
Dynamic Tour Listings: Filterable tour packages fetched from a Django REST API.

Destination Spotlights: Detailed views for various travel locations.

Accommodation Management: integrated lodging options and pricing.

Responsive UI: Mobile-first design using custom CSS and React components.

Lead Generation: Functional contact form for tour inquiries.

**🛠️ Tech Stack**
--

Frontend: React.js, CSS3

Backend: Django Framework

Database: MySQL

Deployment: Vercel (Frontend), Railway/Render (Backend)
